
# OnRight
OnRight는 구직자와 기업 간의 효율적인 매칭 및 기업의 인재 발굴을 지원합니다</br>
![](https://i.imgur.com/weP0NGA.png)


OnRight는 구직자와 기업 모두에게 효율적이고 편리한 채용 프로세스를 제공합니다.</br>
직무 추천 프로그램, 구직자 자소서 및 포트폴리오 등록 & 열람, 기업 분석 등의 서비스를 제공합니다.</br>
